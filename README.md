## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher)
- [Java Development Kit (JDK)](https://openjdk.org/) (v11 or higher)
- [Maven](https://maven.apache.org/) / Gradle (depending on backend config)

### Installation

**1. Clone the repository:**
```bash
git clone https://github.com/Audomsub/Vortex-Admin-Pro.git
cd Vortex-Admin-Pro
```

**2. Setup Frontend:**
```bash
cd frontend
npm install
npm run dev
```

**3. Setup Backend:**
```bash
cd backend/vortex-admin-pro
# Run with Maven or your IDE (e.g. IntelliJ / Eclipse)
./mvnw spring-boot:run
```
